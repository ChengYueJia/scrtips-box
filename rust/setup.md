# Rust Env setup
    Generally on unix/linux system.

## Steps
1. Install rustup
2. Install useful extensions
```text
cargo install cargo-watch
cargo install cargo-edit
rustup component add rls rust-analysis rust-src clippy rustfmt
```
## More about
### cargo install
* cargo-watch
* cargo-edit

### rustup component
* rls 
* rust-analysis 
* rust-src 
* clippy 
* rustfmt
